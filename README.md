
# Portafolio Web

Este repositorio contiene un sitio web personal/portafolio construido con Next.js y TypeScript. Es una página pensada para mostrar proyectos, certificaciones, habilidades y testimonios de manera clara y sencilla.

**Qué encontrarás aquí**
- Página de inicio con héroe, habilidades y testimonios.
- Sección de `Proyectos` con enlaces a cada demo y repositorio.
- Sección de `Certificaciones` / diseños.
- Un footer con enlaces a redes (GitHub, LinkedIn, email).



**Tecnologías principales**
- `Next.js` (framework React) — para el sitio y las rutas.
- `React` + `TypeScript` — componentes tipados.
- `Tailwind CSS` — estilos utilitarios.
- `Contentlayer` / remark / rehype — para procesar contenido y MDX.
- Librerías de apoyo: `swr`, `reading-time`, `rehype-highlight`, entre otras.
